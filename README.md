# deno
My deno Libs


## Denomon

### Use

`deno -A https://raw.githubusercontent.com/lperlaki/deno/master/denomon.js src/main.js`

### Install Bash

`echo "function denomon() { deno run -A https://lperlaki.github.io/deno/denomon.js \$@;}" >> ~/.bashrc && source ~/.bashrc`

#### Use

`denomon src/main.js`
